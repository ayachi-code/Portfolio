Todo

In mijn vrije tijd heb ik een web app gemaakt dat als todo app dient. Ik heb deze webapp gemaakt in pure javascript zonder gebruik te maken van bibliotheken. De todo app werkt als volgt je typte het gene wat je wilt doen in de input en klikt dan op enter. Het gene wat je wilt gaan doen komt dan boven op het scherm te staan. Je kan de todo item verwijderen door op het kruisje te klikken of te dubbelklik. Ik gebruikte mijn todo lijst wel voor een tijde voor persoonelijke zaken. Mijn todo lijst slaat natuurlijk wel de todos lokaal op.
