Secure Programming

Voor mijn eerste specialisatie van het jaar heb ik een interessant vak gevolgd aan de VU met de naam "Secure Programming". Dit vak betrof een introductiecursus op het gebied van computerbeveiliging. Het was de bedoeling dat je tijdens deze cursus leerde hoe je applicaties kon ontwikkelen met het oog op beveiliging. Er werd in het bijzonder gefocust op het identificeren van kwetsbaarheden in software en hoe je deze op een effectieve manier kon oplossen. Dit is een belangrijk aspect van programmeren, vooral omdat steeds meer bedrijven afhankelijk zijn van computersystemen en hun gegevens veilig willen houden. Door deze cursus te volgen, heb ik een waardevolle vaardigheid geleerd die in de toekomst van pas zal komen bij het ontwikkelen van veilige software.
