Videx

Videx is een full-stack webapplicatie bedoeld voor mensen die de indruk hebben films te kijken. De gebruiker kan een specifieke film in de invoer typen, deze geeft informatie over de film. Er zijn 2 zoekopdrachten info zoeken en relatie zoeken. Met de informatiezoekopdracht kun je gegevens over de film ophalen. Het zoeken naar relaties levert daarentegen gerelateerde films op die je zou kunnen bekijken. Ik heb react gebruikt in dit project en een back-end op express. Ik heb mijn eigen API gemaakt die me de informatie gaf over die films. Dit was mijn eerste applicatie waarin ik express en react samen gebruikte. Het was erg leuk om eraan te werken, ik heb veel geleerd met deze module.
