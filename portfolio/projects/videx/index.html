Videx

Videx is a full stack web application ment for preople that have a intression too watch movies. The user can type a sepcific movie in the input, it will give infomation about the movie. There are 2 searches info search and relataion search. With the info seacrh you can retrieve data about the movie.The relation search in the other hand gives related films that you could watch. I used react in this project and a back-end on express. I created my own api that gave me the info I needed. This was my first application where I used express and react together. It was really fun working on it, I learend a lot with this module.
