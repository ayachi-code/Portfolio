Real time chat

Ik heb een realtime chatwebapplicatie gemaakt. Dit was mijn eerste applicatie waarbij ik js-sockets en express gebruikte. Ik heb dit project in mijn vrije tijd gemaakt omdat ik meer wilde leren over netwerken. Deze applicatie heeft me veel geleerd over node js NPM-pakketten. Ik gebruikte mijn kennis ook in nieuwe projecten. Ik heb met veel plezier aan deze applicatie gewerkt. De chat is een wereldwijde chat waarin iedereen over de hele wereld met elkaar kan praten.
