Gamedevelopment blender

Voor de coderclass moesten we een game maken voor een bedrijf.Ik heb een game gemaakt met de blender game engine.De game ging over het verzamelen van voedsel in de steentijd.De game moest online zijn waarbij er 3 grotten zijn.Elke speler heeft een grot en moet dan voedsel verzamelen je moet per dag een x aantal voedsel hebben om de dag te overleven.De game was bedoelt voor mensen een museum.Bij dit project heb ik veel geleerd onder andere het maken van animaties en het werken met blender.
